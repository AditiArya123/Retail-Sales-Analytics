# Python Data Preprocessing

This folder contains the Python preprocessing notebook used before SQL analysis.

## Tasks Performed

- Loaded the raw Superstore dataset using Pandas
- Inspected dataset structure and dimensions
- Checked for missing values
- Checked duplicate records
- Converted date columns
- Standardized column names
- Validated sales and discount values
- Generated summary statistics
- Exported a cleaned dataset for SQL analysis
